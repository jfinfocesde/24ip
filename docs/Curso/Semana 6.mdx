##  Actividad: Creación de un Portafolio Digital

**Objetivo:** Desarrollar un portafolio digital que muestre tus habilidades, experiencias, proyectos y puedas compartir tu información de contacto.

**Pasos:**

**1. Configura tu espacio de trabajo:**

* Crea un nuevo repositorio en GitHub, dándole un nombre descriptivo como "mi-portafolio-digital".
* Clona este repositorio en tu computadora.

**2. Diseña la estructura básica:**

* Crea un archivo "index.html" dentro de la carpeta de tu repositorio.
* Utiliza la etiqueta `<html>` para definir la raíz de tu página web.
* Define la etiqueta `<head>` para añadir información como el título de la página y el conjunto de caracteres (`charset`).
* Define la etiqueta `<body>` para agregar el contenido visible de tu página web.

**3. Agrega secciones con etiquetas semánticas:**

* Crea una sección `<header>` con tu nombre, título (ej: "Portafolio Digital") y un breve lema.
* Agrega una sección `<nav>` para un menú de navegación sencillo con enlaces a secciones como "Sobre mí", "Experiencia", "Proyectos", "Contacto".
* Crea una sección `<main>` que contendrá la información principal de tu portafolio.
* Termina con una sección `<footer>` con información de copyright, enlaces a tus redes sociales y tu información de contacto.

**4. Agrega contenido con etiquetas de formateo:**

* **Sección "Sobre mí":**
    * Utiliza etiquetas de encabezado (`<h1>`, `<h2>`, etc.) para estructurar títulos y subtítulos.
    * Utiliza etiquetas de párrafo (`<p>`) para escribir una breve biografía sobre ti, tus habilidades y áreas de interés.
    * Puedes incluir una imagen de perfil (`<img>`).
* **Sección "Experiencia":**
    * Utiliza una lista con etiquetas `<ul>` y `<li>` para listar tus experiencias laborales o proyectos relevantes.
    * Puedes incluir detalles como el nombre de la empresa, el cargo, el periodo de trabajo y una breve descripción de tus logros.
* **Sección "Proyectos":**
    * Utiliza una lista con etiquetas `<ul>` y `<li>` para listar tus proyectos personales o académicos.
    * Incluye el nombre del proyecto, una breve descripción y enlaces a repositorios en GitHub o sitios web relacionados.
    * Puedes utilizar imágenes (`<img>`) para mostrar ejemplos de tus proyectos.
* **Sección "Contacto":**
    * Incluye un formulario de contacto (`<form>`) con campos para nombre, correo electrónico, mensaje y un botón de envío (`<button type="submit">`).
    * Asegúrate de que el formulario esté configurado para enviar la información a tu correo electrónico.
    * Puedes también incluir enlaces a tus perfiles de redes sociales.

**5. Añade multimedia:**

* Agrega imágenes (`<img>`) relevantes a tus habilidades, proyectos o experiencias.
* Utiliza etiquetas `<video>` o `<audio>` para incluir multimedia si lo deseas.
* Asegúrate de usar el atributo `alt` para proporcionar una descripción alternativa de las imágenes.

**6. Guarda los cambios en el repositorio:**

* Guarda los cambios en tu archivo "index.html".
* Utiliza el comando `git add .` para agregar todos los cambios al área de preparación.
* Utiliza el comando `git commit -m "Mensaje descriptivo del commit"` para guardar los cambios en el historial del repositorio.

**7. Publica en GitHub Pages:**

* En la página de tu repositorio en GitHub, ve a "Settings".
* En la sección "GitHub Pages", configura la rama y carpeta para la publicación.
* Espera unos minutos a que GitHub Pages procese tu sitio web.
* Ahora puedes acceder a tu sitio web en la URL proporcionada.

**Recuerda:**

* Guarda los cambios en tu archivo "index.html" antes de agregarlos al repositorio.
* Revisa la URL de tu sitio web en GitHub Pages para acceder a tu sitio web publicado.

**8. Entrega:**

* Comparte el enlace de tu repositorio de GitHub y de la pagina públicada en el archivo de evidencias de tu carpeta compartida en la semana 6.

:::warning
Nota: No se aceptarán actualizaciones al repositorio después de la fecha y hora de entrega.
:::
