# Semana 11

## Guía Completa de Flexbox

### ¿Qué es Flexbox?
Flexbox (Flexible Box Module) es un modelo de diseño CSS que permite crear diseños flexibles y responsivos. Está diseñado específicamente para distribuir espacio y alinear elementos dentro de un contenedor, incluso cuando sus tamaños son desconocidos o dinámicos.

### Conceptos Básicos

#### 1. Contenedor Flex (Flex Container)
Es el elemento padre que contiene los elementos flex. Se crea usando:
```css
display: flex;
```

#### 2. Elementos Flex (Flex Items)
Son los hijos directos del contenedor flex. Automáticamente se convierten en elementos flexibles.

## Propiedades del Contenedor Flex

### 1. flex-direction
Determina la dirección principal del contenedor:
- `row` (default): izquierda a derecha
- `row-reverse`: derecha a izquierda
- `column`: arriba hacia abajo
- `column-reverse`: abajo hacia arriba

### 2. justify-content
Alinea elementos en el eje principal:
- `flex-start`: elementos al inicio
- `flex-end`: elementos al final
- `center`: elementos al centro
- `space-between`: espacio igual entre elementos
- `space-around`: espacio igual alrededor de elementos
- `space-evenly`: espacio igual entre y alrededor

### 3. align-items
Alinea elementos en el eje transversal:
- `stretch` (default): estira elementos
- `flex-start`: elementos arriba/inicio
- `flex-end`: elementos abajo/final
- `center`: elementos centrados
- `baseline`: alineados por línea base del texto

### 4. flex-wrap
Controla el desbordamiento de elementos:
- `nowrap` (default): una sola línea
- `wrap`: múltiples líneas
- `wrap-reverse`: múltiples líneas en reversa

### 5. gap
Espacio entre elementos:
- `gap`: espacio en ambas direcciones
- `row-gap`: espacio entre filas
- `column-gap`: espacio entre columnas

## Propiedades de los Elementos Flex

### 1. flex-grow
- Determina cuánto puede crecer un elemento
- Valor por defecto: 0
- Valores mayores = más crecimiento

### 2. flex-shrink
- Determina cuánto puede encogerse
- Valor por defecto: 1
- Valores mayores = más encogimiento

### 3. flex-basis
- Tamaño base del elemento
- Puede ser en px, %, auto, etc.

### 4. orden
- Controla el orden de aparición
- Valor por defecto: 0
- Valores negativos permitidos

## Casos de Uso Comunes

1. **Centrado Perfecto**
```css
.container {
    display: flex;
    justify-content: center;
    align-items: center;
}
```

2. **Navegación Responsiva**
```css
.nav {
    display: flex;
    justify-content: space-between;
}
```

3. **Distribución Equitativa**
```css
.item {
    flex: 1;  /* Todos los elementos ocupan el mismo espacio */
}
```

## Ventajas de Flexbox

1. **Simplicidad**
   - Código más limpio y mantenible
   - Menos necesidad de hacks CSS

2. **Responsividad**
   - Adaptable a diferentes tamaños de pantalla
   - Manejo flexible del espacio

3. **Control Bidireccional**
   - Control sobre ejes vertical y horizontal
   - Fácil alineación en ambas direcciones

4. **Orden Dinámico**
   - Reordenamiento visual sin cambiar HTML
   - Útil para diseño responsivo
