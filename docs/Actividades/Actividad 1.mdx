### 3. Actividad: Creando una galería de imágenes responsiva con Flexbox (30%)

Esta actividad te guiará en la creación de una galería de imágenes responsiva utilizando Flexbox en CSS. Aprenderás a estructurar el HTML, aplicar estilos CSS y utilizar las propiedades de Flexbox para crear un diseño adaptable a diferentes tamaños de pantalla.

**Paso 1: Estructura HTML**

1.  **Crea un nuevo archivo HTML:** Abre tu editor de código preferido (como Visual Studio Code) y crea un nuevo archivo HTML, por ejemplo, "galeria.html".

2.  **Estructura básica:** Agrega la siguiente estructura HTML básica al archivo:

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Galería de Imágenes</title>
    <link rel="stylesheet" href="style.css"> 
</head>
<body>
    <div class="galeria">
    </div>
</body>
</html>
```

3.  **Agregar imágenes:** Dentro del `<div>` con la clase "galeria", agrega varios elementos `<img>` para representar las imágenes de tu galería. Asegúrate de incluir los atributos `src` y `alt` para cada imagen.

```html
<div class="galeria">
    <img src="imagen1.jpg" alt="Imagen 1">
    <img src="imagen2.jpg" alt="Imagen 2">
    <img src="imagen3.jpg" alt="Imagen 3">
</div>
```

**Paso 2: Estilos CSS básicos**

1.  **Crea un archivo CSS:** Crea un nuevo archivo CSS llamado "style.css" en la misma carpeta que tu archivo HTML.

2.  **Enlaza el archivo CSS:** En el archivo HTML, dentro de la etiqueta `<head>`, agrega la siguiente línea para enlazar el archivo CSS:

    ```html
    <link rel="stylesheet" href="style.css">
    ```

3.  **Estilos básicos de la galería:** Agrega los siguientes estilos CSS al archivo "style.css" para definir el contenedor de la galería:

```css
.galeria {
    display: flex; 
    flex-wrap: wrap; 
    justify-content: center; 
    gap: 10px; 
    max-width: 800px; 
    margin: 0 auto; 
}
```

**Paso 3: Estilos de las imágenes**

1.  **Ancho responsivo:** Define el ancho de las imágenes para que se ajusten al tamaño de la pantalla utilizando unidades relativas como porcentajes o `vw`.

```css
.galeria img {
    width: 23%; 
    height: auto; 
    object-fit: cover; 
}
```

2.  **Estilos adicionales:** Agrega cualquier estilo adicional que desees para las imágenes, como bordes redondeados, sombras o transiciones.

```css
.galeria img {
    border-radius: 5px;
    box-shadow: 0px 2px 5px rgba(0, 0, 0, 0.2);
    transition: all 0.3s ease;
}
```

**Paso 4: Diseño responsivo con media queries**

1.  **Definir puntos de interrupción:** Decide en qué puntos de interrupción deseas ajustar el diseño de la galería, por ejemplo, para pantallas móviles, tablets y escritorios.

2.  **Agregar media queries:** Utiliza `@media` para definir estilos específicos para cada punto de interrupción.

    *   **Ejemplo:**

    ```css
    @media (max-width: 768px) {
        .galeria img {
            width: 48%; 
        }
    }

    @media (max-width: 480px) {
        .galeria img {
            width: 95%; 
        }
    }
    ```
:::warning[ESTE ES EL PASO MÁS IMPORTANTE]
**Paso 5: Añade comentarios detallados a cada atributo CSS, explicando el estilo que se está aplicando.**

### 2. Actividad: Explorando Propiedades CSS con un Layout de Tarjetas (30%)

**Objetivo:** Esta actividad te guiará paso a paso en la creación de un layout de tarjetas utilizando HTML y CSS, permitiéndote aplicar y comprender diferentes propiedades CSS como `display`, `margin`, `padding`, `position`, `flexbox` y unidades de medida (`px`, `em`, `rem`, `vh`, `vw`, `%`).

**Paso 1: Creación de la Estructura HTML**

1.  **Crea un nuevo archivo HTML:** En tu editor de texto, crea un nuevo archivo y guárdalo como `tarjetas.html`.
2.  **Agrega la estructura básica HTML:**  Inserta el siguiente código HTML en tu archivo `tarjetas.html`:

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Layout de Tarjetas</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="contenedor">
        <div class="tarjeta">
            <img src="imagen1.jpg" alt="Imagen 1">
            <h3>Título de la Tarjeta 1</h3>
            <p>Descripción breve de la tarjeta 1. Puedes agregar más contenido aquí.</p>
        </div>
        <div class="tarjeta">
            <img src="imagen2.jpg" alt="Imagen 2">
            <h3>Título de la Tarjeta 2</h3>
            <p>Descripción breve de la tarjeta 2. Puedes agregar más contenido aquí.</p>
        </div>
        <div class="tarjeta">
            <img src="imagen3.jpg" alt="Imagen 3">
            <h3>Título de la Tarjeta 3</h3>
            <p>Descripción breve de la tarjeta 3. Puedes agregar más contenido aquí.</p>
        </div>
    </div>
</body>
</html>
```

**Paso 2: Creación del Archivo CSS**

1.  **Crea un nuevo archivo CSS:** En la misma carpeta donde guardaste `tarjetas.html`, crea un nuevo archivo llamado `style.css`.

**Paso 3: Estilos Generales del Contenedor**

1.  **Abre `style.css`:**  En tu editor de texto, abre el archivo `style.css`.
2.  **Añade los estilos básicos:**

```css
body {
    font-family: sans-serif;
    margin: 0;
}

.contenedor {
    display: flex; 
    flex-wrap: wrap; 
    justify-content: space-around; 
    padding: 20px;
}
```

**Paso 4: Estilos de las Tarjetas**

1.  **Estiliza las tarjetas:**

```css
.tarjeta {
    width: 300px; 
    margin: 10px; 
    border: 1px solid #ccc; 
    padding: 20px; 
    border-radius: 5px; 
    box-shadow: 0 2px 5px rgba(0, 0, 0, 0.1); 
}
```

**Paso 5: Estilos de las Imágenes**

1.  **Ajusta las imágenes:**

```css
.tarjeta img {
    width: 100%; 
    height: auto; 
    display: block; 
    border-radius: 5px 5px 0 0; 
}
```

**Paso 6: Estilos de los Títulos y Descripciones**

1.  **Estiliza los títulos y descripciones:**

```css
.tarjeta h3 {
    margin-top: 10px; 
}

.tarjeta p {
    font-size: 0.9em; 
    line-height: 1.5; 
}
```
:::warning[ESTE ES EL PASO MÁS IMPORTANTE]
**Paso 7: Añade comentarios detallados a cada atributo CSS, explicando el estilo que se está aplicando.**
:::
